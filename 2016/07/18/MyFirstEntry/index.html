MyFirstEntry

To install hexo on windows was relatively easy, but installing it on a mac system gave me some issues. According to my limited understanding, I think it had to do with installing it on the right directory. Using supo before the hexo command as suggested in some websites did not work. Below are the steps I took to resolve the issue;
I first installed homebrew and xcode on my mac system.
Then I installed node and npm.
Next I installed git using homebrew.
Finally I installed hexo following the instructions on the website below;
www.jr0cket.co.uk/hexo/index.html
The good thing about the website is it walks you through the process of installing and using hexo to publish websites. It goes as far as teaching you how to setup a repository using github.
